# Contributing

Useful contributions here are narrow and test-backed:

- improve deterministic field extraction
- harden file or media handling
- tighten replay and watcher behavior
- clarify Notion mapping behavior
- improve docs without adding fake complexity

Before opening a pull request:

- do not commit real recordings, transcripts, or logs
- do not add hosted LLM dependencies
- keep new environment variables documented in `.env.example`
- add or update tests when behavior changes
